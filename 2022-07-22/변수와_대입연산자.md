# 02. 변수와 대입연산자

자바스크립트에서는 변수를 사용해 특정한 값(데이터)를 저장할 수 있음

```
let myVariable = 5;
```

위에서 알 수 있듯 변수(variable)에 값을 할당할 때 대입연산자(=)를 사용함  
대입연산자는 말 그대로 오른쪽 항에 있는 값을 왼쪽에 있는 변수에 대입하는 역할을 함

```
let myVar =5;
let myNum = myVar;
// 5를 변수 myVar의 값으로 대입
// 그 후 myVar를 myNum 값으로 대입
```

# Assignment

- 변수 a의 값으로 7을 대입
- 변수 b의 값으로 a를 할당
- 변수 b를 console.log로 출력해 값 확인

```
let a = 7;
let b = a;
console.log(b)
// 변수 b의 값은 7이 나옴
```
