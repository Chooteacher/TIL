# Assignment

- length1에는 word1의 길이 값을 할당
- length2에는 word2의 길이 값을 할당
- 직접 계산하지 말고 length를 사용하여 계산
- console.log에서는 두 길이 값의 평균을 구하고 있는 것을 확인
- 두 숫자를 더해서 2로 나눠 평균을 구하기

```
let word1 = "Hello";
let word2 = "World";
let length1 = word1;
let length2 = word2;

console.log((length1.length + length2.length) / 2);
```
